# Turnstile

High-performance distributed rate limiter build with Go, gRPC, and Redis.

## Features

- Token bucket algorithm with atomic Redis operations
- gRPC API for service-to-service communication
- Horizontal scaling with nginx load balancing
- Sub-millisecond latency
- Configurable rate limits per identifier and action

## Status

Work in progress

## Tech Stack

- Go 1.25+
- gRPC & Protocol Buffers
- Redis
- Docker & Docker Compose
